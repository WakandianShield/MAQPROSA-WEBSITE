<p align="center">
Maqprosa-Website is the official corporate website of <b>Maquinados Progresivos de Occidente S.A. de C.V. (MAQPROSA)</b>.
<br><br>
This website was built using HTML, CSS, and JavaScript to create a professional, responsive,
and informative experience focused on showcasing industrial services and precision manufacturing solutions.
</p>

<h2 align="center">Overview</h2>

<p align="center">
Maqprosa-Website is a professional corporate platform designed to present MAQPROSA’s
industrial capabilities in a structured, modern, and technically focused layout.
<br><br>
The project emphasizes clarity, credibility, smooth navigation,
and a well-organized presentation of services, equipment, and company information.
</p>
